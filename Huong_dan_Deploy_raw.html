TÀI LIỆU HƯỚNG DẪN TRIỂN KHAI HỆ THỐNG

(Kèm theo Gói bàn giao Mã thực thi: Ban_Giao_Ma_Thuc_Thi_SongHan_Obfuscated.zip)

Kính gửi: Ban Điều Hành / Bộ phận IT của Hội Xây Dựng Sông Hàn

Tài liệu này hướng dẫn cách đưa hệ thống Phần mềm Web App Quản lý KPI lên môi trường mạng Internet từ file nén mã thực thi (Deployment Build) đã được Đơn vị phát triển bàn giao.

LƯU Ý BẢO MẬT: File mã thực thi bàn giao là bản Code đã được biên dịch, tối ưu hóa và mã hóa (Obfuscated) nhằm đảm bảo an toàn tuyệt đối, chống dịch ngược và đánh cắp dữ liệu kinh doanh của hệ thống. File này không thể đọc hiểu hay chỉnh sửa bằng các phần mềm soạn thảo thông thường, nhưng hoạt động hoàn hảo 100% khi được đưa lên máy chủ.

1. YÊU CẦU MÁY CHỦ (SERVER / HOSTING)

Do hệ thống được phát triển theo kiến trúc SPA (Single Page Application) công nghệ mới nhất, hệ thống không đòi hỏi máy chủ cấu hình quá phức tạp. Chỉ cần đáp ứng các điều kiện sau:

  • Máy chủ Web Hosting thông thường (cPanel / DirectAdmin) hoặc các nền tảng Cloud hiện đại (Google Firebase, Vercel, Netlify).
  • Không cần cài đặt PHP hay MySQL (Dữ liệu đã được liên kết với Cloud Database riêng).
  • Có hỗ trợ chuyển hướng (URL Rewrite / Fallback) về file index.html.

2. HƯỚNG DẪN ĐƯA LÊN MẠNG BẰNG WEB HOSTING (CPANEL)

Đây là cách phổ biến nhất nếu Hội đã có sẵn một Web Hosting và Tên miền riêng (ví dụ: kpi.hoisonghan.com).

Bước 1: Giải nén file bàn giao

  1. Tải file Ban_Giao_Ma_Thuc_Thi_SongHan_Obfuscated.zip về máy tính.
  2. Giải nén file này ra. Bên trong bạn sẽ thấy các tệp tin cơ bản như: index.html, thư mục assets (chứa các file mã hóa .js, .css).

Bước 2: Tải lên máy chủ (Upload)

  1. Đăng nhập vào trình quản lý Hosting (cPanel hoặc DirectAdmin) của Hội.
  2. Truy cập vào mục File Manager (Quản lý Tệp).
  3. Mở thư mục gốc của tên miền (thường là public_html).
  4. Upload toàn bộ các file và thư mục vừa giải nén (bao gồm index.html và assets) vào thẳng thư mục public_html này.

Bước 3: Cấu hình điều hướng (Rất quan trọng) Vì là ứng dụng SPA, bạn cần tạo một file tên là .htaccess (nằm cùng chỗ với file index.html vừa upload) và dán đoạn mã sau vào để hệ thống không bị lỗi "Not Found" khi F5 (Tải lại trang):

<IfModule mod_rewrite.c>
  RewriteEngine On
  RewriteBase /
  RewriteRule ^index\.html$ - [L]
  RewriteCond %{REQUEST_FILENAME} !-f
  RewriteCond %{REQUEST_FILENAME} !-d
  RewriteRule . /index.html [L]
</IfModule>

Bước 4: Hoàn tất Truy cập vào tên miền của bạn bằng trình duyệt. Hệ thống đã sẵn sàng hoạt động!

Nếu Hội chưa có máy chủ và muốn đưa lên mạng miễn phí, an toàn nhất:

  1. Truy cập trang web: https://app.netlify.com/drop
  2. Đăng nhập bằng tài khoản Github hoặc Email.
  3. Cầm thư mục đã được giải nén từ file Ban_Giao_Ma_Thuc_Thi_SongHan_Obfuscated.zip và Kéo Thả (Drag & Drop) thẳng vào vòng tròn trên web Netlify.
  4. Netlify sẽ tự động cung cấp một đường link để truy cập hệ thống ngay lập tức (Bạn có thể trỏ tên miền riêng vào đường link này sau).

Nếu có bất kỳ khó khăn nào trong quá trình đưa hệ thống lên mạng, vui lòng liên hệ với Đơn vị phát triển (Tân Gia Khang) để được hỗ trợ chuyên sâu.
